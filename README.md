# 🌍 Women’s Global Development & Wellbeing Analysis (EDA Project)

![Banner](women.png)

An interactive **Exploratory Data Analysis (EDA) dashboard** built using **Python, Pandas, and Matplotlib**, designed to visualize and understand global patterns in women’s wellbeing across countries.

---

## ✨ Overview

This project explores how different socio-economic and development factors influence women’s wellbeing globally.

It transforms raw dataset values into an **interactive visual storytelling dashboard**.

Instead of static charts, users can click through different indicators and instantly view:

- Distributions  
- Summary statistics  
- Comparisons

## 🎯 Research Focus

How do education, financial autonomy, healthcare access, technology ownership, and wealth vary across countries in relation to women’s wellbeing?

- 📚 Education levels (average years of schooling)  
- 💰 Financial decision-making power

- 📊 Wealth index distribution

## 🚀 How It Works

- Dataset is loaded using Pandas  
- Missing values are cleaned  
- Variables are mapped to human-readable labels  
- A dashboard window is created using Matplotlib  
- Buttons allow users to interactively explore variables  
- Clicking a button generates a histogram with:
  - Distribution of values  
  - Mean & median statistics  
  - Clear axis labeling with real-world meaning
